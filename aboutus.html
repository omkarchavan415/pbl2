Live Weather Data

Interactive charts for temperature trends, humidity, wind speed, and simulated pressure data.

Dynamic Background

The background changes automatically based on the current weather condition.

Interactive Weather Map

Click anywhere on the map to see weather details using Leaflet and OpenStreetMap.

Login Authentication

Simple login page with hardcoded credentials for access control to the main app.
